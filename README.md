# Customer Churn Analysis

## Overview

Analyzed customer churn behavior using Excel, SQL, Python, and Tableau.

## Tools Used

* Excel
* MySQL
* Python (Pandas, Matplotlib, Seaborn)
* Tableau

## Key Insights

* Month-to-month customers have the highest churn.
* Fiber optic users show higher churn than DSL users.
* Electronic check customers have the highest churn.
* Long-term contracts improve customer retention.

## Project Components

* Excel Dashboard
* SQL Analysis
* Python EDA & Visualization
* Tableau Dashboard

## Dataset

Telco Customer Churn Dataset
